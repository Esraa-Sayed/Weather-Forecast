# Weather-Forecast
Android Mobile Application that displays the weather status and the temperature
from your location. Also, you can pick a specific location on the map and/or search for it by auto-complete edit text and add it to a list of favorite locations and get all the weather information at this location. In addition you can set an alert for rain, wind, very low or very high temperature, fog, snow â€¦ etc.

## Project functionality:
  * MVVM
  * Room
  * Retrofit
  * Gson
  * Coroutine
  * Coroutine worker
  * Notification
  * Glide
  * Navigation component
  * LBS(Location Based Service)
  * minSdk 27
  * targetSdk 32
